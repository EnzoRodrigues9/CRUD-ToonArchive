O ToonArchive é um um site de postagem e leitura de quadrinhos e, para esse CRUD, foi feito a tela de cadastro de usuário do site. 
O front end é uma tela característica, intuitiva feita em angular que permite o usuário colocar informações de login, senha e email. Além disso, exibe uma lista de usuários cadastrados abaixo do botão "Cadastrar-se" que se atualiza em tempo real.
O back end é feito em Spring boot e possui um armazenamento temporário de informações do front end em h2, fazendo a parte de salvamento das informações que o usuário cadastra e ajudando na atualização da lista de usuários cadastrados.
